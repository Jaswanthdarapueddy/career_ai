# CareerAI – AI-Powered Career Guidance & Employment Readiness System

## Slide 1: Title
**CareerAI**  
AI-Powered Career Guidance & Employment Readiness System  
Aligned with SDG-8: Decent Work and Economic Growth

---

## Slide 2: Problem Statement
- Many individuals lack personalized career guidance.
- Resumes often fail ATS filters.
- Skill gaps are unclear.
- No clear roadmap to employment readiness.

---

## Slide 3: Solution Overview
**CareerAI** provides:
- Profile-based AI analysis
- Career role recommendations
- ATS-friendly resume generation
- Skill gap identification
- Certification and project suggestions
- Structured development roadmap

---

## Slide 4: Tech Stack
- **Frontend**: React (Vite), Lucide React icons, glassmorphism UI
- **Backend**: Python FastAPI
- **AI/Workflow**: Relay.app (Agentic AI automation)
- **Database**: Supabase (PostgreSQL)

---

## Slide 5: Architecture
```
Frontend (React/Vite) → API Proxy → FastAPI → Relay.app (AI Workflow) → Supabase
```
- Secure, scalable, and modular design.

---

## Slide 6: Core Features
1. User profile input
2. AI career analysis
3. Resume generation
4. Skill gap detection
5. Certification/project suggestions
6. Development roadmap
7. Secure data storage

---

## Slide 7: UI/UX Design
- Minimal, professional glassmorphism theme
- Smooth animations and transitions
- Responsive design
- Accessible and intuitive flow

---

## Slide 8: Relay.app Workflow
1. **Manual Trigger**: Accepts user profile JSON
2. **Career Analysis Agent**: Recommends roles
3. **Resume Generation Agent**: Creates ATS resume
4. **Skill & Readiness Agent**: Identifies gaps and roadmap
5. **Final Output**: Returns structured JSON

---

## Slide 9: Database Schema
- `user_profiles`: Stores user input data
- `ai_outputs`: Stores AI-generated recommendations
- Privacy and integrity ensured
