# Career AI Demo

This is a simple demonstration of a career AI application that takes user input and generates a career plan.

## How to Run

1. Open the `career-ai-demo` directory.
2. Open `index.html` in your web browser.
3. Fill out the form and click "Submit".
4. The application will then show a loading screen and then the results.

## Screens

### Screen 1: User Input Form (`index.html`)

- Collects the user's full name, education, skills, career interest, and experience level.

### Screen 2: AI Processing Screen (`index2.html`)

- Shows a loading indicator and a message that the user's profile is being analyzed.
- This is a simulation and has a 3-second delay.

### Screen 3: Result / Output Screen (`index2.html`)

- Displays the AI-generated results after the processing is complete.
- The results include:
    - Career Recommendation
    - Generated Resume (placeholder)
    - Certificates & Projects (placeholder)
    - Skill Roadmap (placeholder)
